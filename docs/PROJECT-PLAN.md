## 2. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| **프레임워크** | Astro 5.x | 콘텐츠 중심, JS 제로 출력, AdSense 최적 호환 |
| **스타일링** | Tailwind CSS 4 | 유틸리티 기반, 빠른 개발, 최적 번들 |
| **콘텐츠** | Markdown/MDX | 파일 기반 관리, CMS 불필요, Git 버전관리 |
| **배포** | Vercel (무료 티어) | 글로벌 CDN, 자동 HTTPS, 제로 설정 |
| **도메인** | 커스텀 도메인 필수 | AdSense 승인 필수 요건 |
| **분석** | Google Analytics 4 | 트래픽 분석, Search Console 연동 |
| **폰트** | Pretendard | 한국어 최적화 웹폰트 |

### 핵심 기술 선정 이유: Astro

1. **순수 정적 HTML** 출력 → AdSense 광고 스크립트 완벽 동작
2. **Core Web Vitals 100점** 근접 → 검색 순위 유리
3. **Markdown 네이티브** → 글 작성에만 집중
4. **빌드 속도 빠름** → 개발 생산성 극대화
5. **필요 시 React 컴포넌트** Islands 방식으로 부분 도입 가능

## 3. 사이트 구조 & URL 설계

### 페이지 구조

```
/                              → 홈 (최신 글 + 카테고리 소개)
/blog/                         → 전체 글 목록
/blog/[slug]/                  → 개별 포스트
/category/saving/              → 저축/예금 카테고리
/category/investing/           → 투자 입문 카테고리
/category/tax/                 → 세금/연말정산 카테고리
/category/tools/               → 재테크 도구 리뷰 카테고리
/category/tips/                → 생활비 절약 카테고리
/about/                        → 소개 (필수)
/privacy-policy/               → 개인정보처리방침 (필수)
/contact/                      → 연락처 (필수)
/terms/                        → 이용약관
/disclaimer/                   → 면책조항
```

### SEO 설정

- `sitemap.xml` 자동 생성 (`@astrojs/sitemap`)
- `robots.txt` 사이트맵 경로 명시
- 모든 페이지 고유 `<title>` + `<meta description>`
- Open Graph / Twitter Card 메타 태그
- JSON-LD 구조화 데이터 (BlogPosting, WebSite, BreadcrumbList)
- `hreflang="ko"` 한국어 지정
- canonical URL 설정
