## 1. AdSense 승인을 위한 콘텐츠 전략

### 1.1 최소 필요 글 수 및 글당 권장 글자 수

| 항목 | 권장 기준 | 비고 |
|------|----------|------|
| **최소 글 수** | 20~30개 | Google은 공식 최소치를 명시하지 않지만, 15개 미만은 거절 위험이 높음 |
| **글당 최소 글자 수** | 1,500자 이상 (한글 기준) | 영문 기준 800~1,500 words, 한글은 1,500~3,000자 권장 |
| **심층 콘텐츠** | 3,000~5,000자 | 카테고리당 2~3개의 심층 글(pillar content) 필요 |
| **도메인 나이** | 최소 3~6개월 | 새 도메인도 우수한 콘텐츠로 승인 가능하나 안전하게 3개월 이상 권장 |

### 1.2 콘텐츠 품질 기준 (2026년 강화 사항)

- **100% 오리지널 콘텐츠**: AI 생성 콘텐츠를 그대로 사용하면 거절 위험이 높음. 반드시 사람이 검수하고 독자적 관점 추가
- **E-E-A-T 원칙 준수**: 경험(Experience), 전문성(Expertise), 권위성(Authoritativeness), 신뢰성(Trustworthiness)
- **사용자 의도 충족**: 검색 의도에 정확히 부합하는 답변 제공
- **광고 목적이 아닌 가치 제공 목적**: Google은 "광고 수익만을 위한 사이트"를 적극 거절

### 1.3 카테고리 구조 (3~5개 카테고리)

#### 니치 A: 개인 재테크/금융 정보
```
홈
├── 저축/예금 전략
├── 투자 기초
├── 생활비 절약 팁
├── 세금/연말정산
└── 재테크 도구 리뷰
```

#### 니치 B: 개발자/IT 기술 블로그
```
홈
├── 프로그래밍 튜토리얼
├── 개발 도구 & 환경
├── 웹 개발 가이드
├── 커리어 & 취업
└── 기술 트렌드
```

#### 니치 C: 생활 정보/라이프스타일 팁
```
홈
├── 건강 & 웰빙
├── 살림 & 정리 팁
├── 요리 & 레시피
├── 여행 & 나들이
└── 생활 꿀팁
```

### 1.4 콘텐츠 캘린더 (초기 30개 글 계획)

> AdSense 신청 전까지 30개 글을 4~6주에 걸쳐 발행합니다.
> 하루 1개 또는 격일 1개 발행 속도를 유지합니다.

| 주차 | 발행 수 | 콘텐츠 유형 | 비고 |
|------|---------|------------|------|
| 1주차 | 5~7개 | 필수 정적 페이지 + 카테고리별 기본 글 | About, 개인정보처리방침, 면책조항, 연락처 포함 |
| 2주차 | 7~8개 | 각 카테고리 심층 글 (pillar content) | 카테고리당 1~2개 핵심 콘텐츠 |
| 3주차 | 7~8개 | 롱테일 키워드 타겟 글 | 내부 링크로 pillar 글과 연결 |
| 4주차 | 5~7개 | 실용적 가이드/리스트 글 | "방법", "비교", "추천" 형식 |
| 5~6주차 | 3~5개 | 보충 콘텐츠 + 기존 글 업데이트 | AdSense 신청 준비 |

## 2. SEO 최적화 콘텐츠 작성법

### 2.1 제목 태그 (Title Tag) 최적화

```html
<!-- 권장 형식 -->
<title>핵심 키워드 - 보조 설명 | 사이트명</title>

<!-- 좋은 예 -->
<title>2026년 연말정산 환급 많이 받는 방법 7가지 | 머니가이드</title>

<!-- 나쁜 예 -->
<title>연말정산</title>
```

**규칙:**
- 길이: 50~60자 (한글 기준 25~30자)
- 핵심 키워드를 앞쪽에 배치
- 숫자 포함 시 클릭률(CTR) 향상 (예: "7가지", "2026년")
- 사이트명은 파이프(|) 또는 대시(-) 뒤에 배치
- H1 태그와 동일하거나 유사하게 작성

### 2.2 메타 디스크립션 작성법

```html
<meta name="description" content="2026년 연말정산에서 환급을 극대화하는 7가지 핵심 전략을 소개합니다. 소득공제, 세액공제 항목별 체크리스트와 실제 계산 예시를 확인하세요.">
```

**규칙:**
- 길이: 150~160자 (한글 기준 70~80자)
- 핵심 키워드 자연스럽게 포함
- 행동 유발 문구 포함 ("확인하세요", "알아보세요", "비교해보세요")
- 각 페이지마다 고유한 디스크립션 작성
- 요약이 아닌, 읽고 싶게 만드는 문장으로 작성

### 2.3 내부 링크 전략

#### 토픽 클러스터 모델

```
[Pillar 콘텐츠: 2026 재테크 완벽 가이드]
         │
    ┌────┼────┬────────┐
    ▼    ▼    ▼        ▼
[예금]  [주식] [부동산]  [절약]
 기초   기초   기초     팁
```

**내부 링크 규칙:**
- 각 글에 최소 3~5개 내부 링크 포함
- Pillar 글에서 클러스터 글로, 클러스터 글에서 Pillar 글로 양방향 연결
- 앵커 텍스트는 설명적이고 키워드를 포함 ("여기를 클릭하세요" 금지)
- 관련성 높은 콘텐츠끼리만 연결
- 모든 페이지는 홈에서 3클릭 이내 도달 가능하도록 설계
- 카테고리 간 교차 링크도 자연스럽게 활용 (월 1~2회)

### 2.4 이미지 ALT 태그

```html
<!-- 좋은 예 -->
<img src="tax-refund-chart.webp"
     alt="2026년 연말정산 소득공제 항목별 환급액 비교 차트"
     width="800" height="450"
     loading="lazy">

<!-- 나쁜 예 -->
<img src="img001.jpg" alt="이미지">
<img src="chart.png" alt="">
```

**규칙:**
- 모든 이미지에 설명적 ALT 텍스트 필수
- 키워드를 자연스럽게 포함하되 키워드 스터핑 금지
- 이미지 파일명도 설명적으로 작성 (예: `tax-refund-chart.webp`)
- WebP 포맷 우선 사용 (로딩 속도 향상)
- width, height 속성 명시 (CLS 방지)
- `loading="lazy"` 적용 (성능 최적화)

## 4. 필수 정적 페이지 콘텐츠

### 4.1 About (소개) 페이지

```markdown
## 필수 포함 항목

1. **사이트 소개**: 사이트의 목적, 미션, 제공하는 가치
2. **운영자 소개**: 이름(또는 필명), 간략한 경력/전문성
3. **전문성 증명**: 해당 분야 경험, 자격증, 관련 학력 등 (E-E-A-T)
4. **사이트 히스토리**: 사이트 시작 배경, 목표
5. **연락 방법**: 이메일 또는 문의 폼 링크
6. **프로필 사진**: 신뢰도를 높이는 실제 사진 (선택사항이지만 권장)
```

**예시 구조:**

```
[사이트명]에 오신 것을 환영합니다

[사이트명]은 [대상 독자]를 위한 [주제] 정보를 제공하는 블로그입니다.

## 운영자 소개
안녕하세요, [이름]입니다. 저는 [관련 경력/경험]을 바탕으로
[분야]에 대한 실용적이고 검증된 정보를 공유하고 있습니다.

## 사이트 목적
[사이트명]은 다음과 같은 정보를 제공합니다:
- [카테고리 1] 관련 실전 가이드
- [카테고리 2] 최신 트렌드와 분석
- [카테고리 3] 초보자를 위한 입문 콘텐츠

## 연락처
궁금한 점이 있으시면 [이메일주소]로 연락해주세요.
```

### 4.2 개인정보처리방침 필수 항목

> Google AdSense 승인을 위해 반드시 필요하며, 개인정보처리방침 URL을 AdSense 계정에 등록해야 합니다.

```markdown
## 필수 포함 항목

1. **개인정보의 처리 목적**
   - 수집하는 개인정보 항목 명시
   - 처리 목적 (방문 분석, 광고 제공 등)

2. **개인정보 수집 항목 및 방법**
   - 자동 수집 항목: IP 주소, 쿠키, 방문 기록 등
   - 수집 방법: 웹사이트 방문 시 자동 수집

3. **쿠키(Cookie) 사용 안내** ★ AdSense 필수
   - Google 및 제3자 광고 네트워크의 쿠키 사용 고지
   - DoubleClick DART 쿠키 관련 설명
   - 쿠키 거부/삭제 방법 안내

4. **제3자 광고 서비스** ★ AdSense 필수
   - Google AdSense 사용 고지
   - 제3자가 쿠키를 삽입하거나 웹 비콘으로 정보를 수집할 수 있음을 고지

5. **개인정보의 보유 및 파기**
   - 보유 기간 및 파기 방법

6. **이용자의 권리**
   - 개인정보 열람, 수정, 삭제 요청 방법
   - 쿠키 설정 변경 방법

7. **개인정보 보호책임자**
   - 담당자 이름 또는 연락처

8. **개인정보처리방침 변경**
   - 변경 시 고지 방법
   - 시행일자

9. **한국 개인정보 보호법 관련** (한국 사이트의 경우)
   - 개인정보 보호법에 따른 고지 의무 사항
```

### 4.3 면책조항 (Disclaimer) 필수 항목

## 필수 포함 항목

    전문적인 [법률/의료/재무] 조언을 대체하지 않습니다."

   "최대한 정확한 정보를 제공하기 위해 노력하지만,
    정보의 완전성, 정확성, 신뢰성을 보장하지 않습니다."

3. **외부 링크 면책**
   "본 사이트에 포함된 외부 링크는 편의를 위해 제공되며,
    외부 사이트의 콘텐츠에 대한 책임은 지지 않습니다."

4. **광고 및 제휴 공개** ★ AdSense/제휴마케팅 필수
   "본 사이트는 Google AdSense를 통한 광고를 게재하고 있으며,
    일부 콘텐츠에는 제휴 링크가 포함될 수 있습니다.
    제휴 링크를 통한 구매 시 사이트 운영자에게 소정의 수수료가
    지급될 수 있습니다."

5. **콘텐츠 사용 제한**
   "본 사이트의 콘텐츠는 저작권법에 의해 보호됩니다.
    무단 복제, 배포를 금합니다."

6. **변경 사항**
   "본 면책조항은 사전 통지 없이 변경될 수 있습니다."

### 4.4 연락처 (Contact) 페이지

```markdown
## 필수 포함 항목

1. **연락 목적 안내**
   - 문의 가능한 사항 안내 (콘텐츠 제안, 오류 신고, 협업 문의 등)

2. **연락 방법** (최소 1개 이상)
   - 이메일 주소 (가장 기본)
   - 문의 폼 (Contact Form) - 권장
   - SNS 계정 (선택)

3. **응답 시간 안내**
   - 예상 응답 시간 (예: "영업일 기준 2~3일 내")

4. **문의 폼 필드** (폼 사용 시)
   - 이름
   - 이메일
   - 제목
   - 메시지 내용
   - 개인정보 수집 동의 체크박스
```

## 5. AdSense 승인 전 체크리스트

### 기술적 요구사항
- [ ] HTTPS/SSL 인증서 설치
- [ ] 유료 도메인 사용 (무료 서브도메인 금지)
- [ ] 모바일 반응형 디자인
- [ ] 페이지 로딩 속도 3초 이내
- [ ] sitemap.xml 생성 및 Google Search Console 등록
- [ ] robots.txt 적절히 설정
- [ ] Google Analytics 연동

### 콘텐츠 요구사항
- [ ] 최소 20~30개 오리지널 글 발행
- [ ] 각 글 1,500자 이상 (한글 기준)
- [ ] 카테고리별 균등한 글 분배 (각 카테고리 최소 4~5개)
- [ ] 모든 글에 적절한 이미지 포함 (저작권 프리 이미지)
- [ ] 내부 링크 구조 완성

### 필수 페이지
- [ ] About (소개) 페이지
- [ ] 개인정보처리방침 페이지
- [ ] 면책조항 페이지
- [ ] 연락처 (Contact) 페이지

### SEO 최적화
- [ ] 모든 페이지에 고유한 title 태그
- [ ] 모든 페이지에 고유한 meta description
- [ ] 모든 이미지에 alt 태그
- [ ] Schema.org 구조화 데이터 적용 (BlogPosting, WebSite, BreadcrumbList)
- [ ] 내부 링크 3~5개씩 포함
- [ ] URL 구조 깔끔하게 설정 (한글 slug 또는 영문 slug)

### 금지 사항
- [ ] 저작권 침해 콘텐츠 없음
- [ ] 성인/폭력/도박 등 정책 위반 콘텐츠 없음
- [ ] 자동 생성 콘텐츠 없음 (AI 그대로 복사 금지)
- [ ] 다른 사이트 콘텐츠 복사/스크래핑 없음
- [ ] 과도한 광고 배치 없음 (승인 전에는 다른 광고 최소화)
