## 1. Google AdSense 승인 필수 요구사항 (2025-2026 최신 기준)

### 1.1 최소 콘텐츠 양

| 항목 | 권장 기준 |
|------|-----------|
| 최소 게시글 수 | 15~25개 (최소 10개 이상) |
| 게시글 당 글자 수 | 800~1,500단어 이상 (한글 기준 약 1,600~3,000자) |
| 콘텐츠 특성 | 독창적, 인간이 작성한 글 (AI 단순 생성 글은 거절 사유) |
| 게시 주기 | 정기적 업데이트 (최소 주 2~3회 권장) |

**핵심**: 구글은 양보다 **품질**을 중시합니다. 10개의 고품질 아티클이 50개의 저품질 글보다 낫습니다.

### 1.2 필수 페이지 (미구비 시 심사 자동 중단)

1. **소개(About) 페이지**: 운영자 신원, 사이트 목적 명시
2. **개인정보처리방침(Privacy Policy)**: 방문자 데이터 수집/이용 방식 구체 기술, 쿠키 사용 고지
3. **문의(Contact) 페이지**: 실제 동작하는 이메일 또는 문의 양식 포함
4. **면책조항(Disclaimer)**: 콘텐츠 책임 범위 명시 (권장)
5. **이용약관(Terms of Service)**: 서비스 이용 조건 (권장)

**참고**: 1~3번은 **필수**, 4~5번은 **강력 권장**입니다.

### 1.3 콘텐츠 품질 기준

- **E-E-A-T 신호**: Experience(경험), Expertise(전문성), Authoritativeness(권위), Trustworthiness(신뢰성)
- **독창성**: 복사/붙여넣기 또는 스핀 콘텐츠 금지
- **가치 제공**: 방문자에게 실질적 도움이 되는 정보
- **AI 콘텐츠 주의**: AI로 생성한 글을 그대로 게시하면 거절될 수 있음. 반드시 인간의 편집/검증 필요
- **결정을 도와주는 콘텐츠**: 단순 정보 나열이 아닌, 비교/분석/추천 등 의사결정 지원

### 1.4 기술적 요구사항

| 항목 | 요구 사항 |
|------|-----------|
| SSL 인증서 | HTTPS **필수** (HTTP 사이트 승인 불가) |
| 모바일 반응형 | 모바일 친화적 디자인 필수 |
| 페이지 로딩 속도 | 4초 이내 (6초 이상 시 거절 가능) |
| 사이트 네비게이션 | 명확하고 직관적인 메뉴 구조 |
| 사이트맵 | sitemap.xml 제출 권장 |
| Google Search Console | 등록 및 인덱싱 확인 필수 |

### 1.5 도메인/사이트 나이 요건

- **공식 최소 요건**: 없음 (구글 공식 정책상 명시된 최소 기간 없음)
- **실질적 권장**: 도메인 등록 후 최소 2~4주, 이상적으로는 **3~6개월**
- **신규 도메인**: 양질의 콘텐츠가 있으면 1~2개월 내에도 승인 가능
- **Google Search Console에 인덱싱**된 상태여야 함

### 1.6 트래픽 요건

- **공식 최소 트래픽 요건**: **없음**
- **실질적 권장**: 일일 100~500명 방문자가 있으면 승인 확률 상승
- **핵심**: 트래픽보다 콘텐츠 품질과 사이트 구조가 더 중요
- 유기적(검색) 트래픽이 유료 트래픽보다 유리

### 1.7 금지 콘텐츠 (위반 시 승인 거절 또는 계정 정지)

| 카테고리 | 세부 내용 |
|----------|-----------|
| **불법 콘텐츠** | 불법 활동 조장, 위조 문서, 불법 약물 |
| **저작권 침해** | 무단 복제 오디오/비디오, 소프트웨어, 문학작품 |
| **폭력/혐오** | 인종, 종교, 장애 등 기반 차별/비하 콘텐츠 |
| **성인 콘텐츠** | 성적으로 노골적인 콘텐츠 |
| **위험 콘텐츠** | 무기, 폭발물, 자해 관련 콘텐츠 |
| **기만적 콘텐츠** | 허위 정보, 클릭베이트, 오도하는 콘텐츠 |
| **프레이밍** | 타 사이트 콘텐츠를 프레임 내에 무단 표시 |
| **온라인 도박** | 허용 국가 외 지역 대상 도박 콘텐츠 (한국은 허용 국가 목록에 포함) |
| **담배/주류** | 담배 제품 판매/홍보, 과도한 음주 조장 |

### 1.8 승인 심사 기간

- 평균 **3~14일** (최대 4주)
- 최근 추세: 양질의 사이트는 **1~3일** 내 승인 사례 증가

## 2. 2026년 기준 수익성 높은 블로그/사이트 니치 분석

### 2.1 한국 AdSense 시장 개요

| 지표 | 수치 |
|------|------|
| 한국 평균 CPC | **$0.10** (약 130원) |
| 미국 평균 CPC | $0.61 |
| 일본 평균 CPC | $0.14 |
| 싱가포르 평균 CPC | $0.27 |

**시사점**: 한국의 CPC는 글로벌 대비 낮은 편이므로, **높은 트래픽 확보** 또는 **고단가 니치 집중**이 필수입니다.

### 2.2 한국어 콘텐츠 고CPC 분야 (Top 5)

#### 1위: 금융/보험 (CPC: 500~5,000원+)
- **세부 주제**: 신용대출, 자동차보험 비교, 생명보험, 청년 전세대출, 주택담보대출
- **특징**: 가장 높은 CPC, 광고주 경쟁 극도로 치열
- **검색량**: 매우 높음 (금융 관련 검색은 연중 꾸준)
- **예시 키워드**: "2026년 청년 전세대출 조건", "자동차보험 비교 사이트"

#### 2위: 법률 서비스 (CPC: 1,000~10,000원+)
- **세부 주제**: 교통사고 합의금, 이혼 변호사, 개인회생, 파산 신청
- **특징**: 건당 광고 단가가 매우 높음
- **검색량**: 중간 (이벤트성 검색 많음)
- **예시 키워드**: "교통사고 합의금 계산", "개인회생 자격 조건"

#### 3위: IT/테크놀로지 (CPC: 200~2,000원)
- **세부 주제**: AI 도구 리뷰, SaaS 비교, 사이버보안, 코딩 교육
- **특징**: 꾸준히 성장하는 분야, 영문 콘텐츠와 시너지 가능
- **검색량**: 높음 (기술 발전에 따라 지속 증가)
- **예시 키워드**: "ChatGPT 대안 AI 도구 비교", "클라우드 호스팅 추천"

#### 4위: 건강/의료 (CPC: 300~3,000원)
- **세부 주제**: 건강보험, 영양제 비교, 다이어트, 질병 예방
- **특징**: YMYL(Your Money Your Life) 분야로 E-E-A-T 매우 중요
- **검색량**: 매우 높음 (계절별 변동 있음)
- **주의**: 의료 전문성 없이 건강 주장 시 정책 위반 위험

#### 5위: 부동산 (CPC: 500~5,000원)
- **세부 주제**: 아파트 시세, 청약 정보, 부동산 세금, 인테리어
- **특징**: 한국 특성상 부동산 관심도 극도로 높음
- **검색량**: 매우 높음
- **예시 키워드**: "2026년 청약 일정", "아파트 취득세 계산"

### 2.3 검색 트래픽이 풍부한 분야

| 분야 | 월간 검색량 수준 | 시즌성 |
|------|-----------------|--------|
| 생활정보/정책 | 매우 높음 | 정책 변경 시 급증 |
| 재테크/절약 | 높음 | 연중 꾸준 |
| IT/테크 리뷰 | 높음 | 신제품 출시 시 급증 |
| 건강/운동 | 높음 | 봄/여름 급증 |
| 여행/맛집 | 높음 | 계절성 있음 |
| 교육/자격증 | 중간~높음 | 시험 시즌 급증 |

### 2.4 경쟁 강도 분석

| 분야 | 경쟁 강도 | 진입 난이도 | 비고 |
|------|-----------|-------------|------|
| 금융/보험 | 극심 | 높음 | 대형 금융사 블로그와 경쟁 |
| 법률 | 높음 | 높음 | 법률사무소 마케팅과 경쟁 |
| IT/테크 | 중간~높음 | 중간 | 개인 블로거가 경쟁 가능 |
| 건강/의료 | 높음 | 높음 | E-E-A-T 증명 필요 |
| 생활정보 | 중간 | 낮음 | 진입 쉬우나 CPC 낮음 |
| 재테크/절약 | 중간~높음 | 중간 | 개인 경험 기반 차별화 가능 |

### 2.5 장기적 가치 분석

**장기 가치 높음 (에버그린 콘텐츠)**:
- 개인 재무 관리, 절약 팁
- IT 기초 교육, 프로그래밍 가이드
- 건강 관리 기본 정보
- 생활 법률 상식

**트렌드 의존적 (정기 업데이트 필요)**:
- 정부 정책/지원금 정보
- 신제품 리뷰
- 부동산 시세/청약

## 3. 추천 사이트 주제 3가지

### 추천 1: "실용 재테크 & 개인 금융 가이드"

**컨셉**: 2030세대 직장인을 위한 실질적인 돈 관리 가이드. 대출 비교, 절세 전략, 저축 방법, 투자 입문 등을 다루는 개인 금융 전문 사이트.

| 구분 | 내용 |
|------|------|
| **장점** | - CPC가 가장 높은 분야 (금융/보험 키워드) |
| | - 연중 꾸준한 검색 수요 |
| | - 정부 정책 변경 시 트래픽 급증 기회 |
| | - 제휴 마케팅(카드/대출 비교)으로 추가 수익 가능 |
| | - 롱테일 키워드 풍부 ("2026년 청년 전세대출 조건 및 신청 방법") |
| **단점** | - 경쟁이 매우 치열 (대형 금융사, 핀테크 기업 블로그) |
| | - 정확한 금융 정보 필요 (오류 시 법적 리스크) |
| | - YMYL 분야로 E-E-A-T 증명 중요 |
| | - 콘텐츠 정기 업데이트 필수 (금리/정책 변동) |
| **예상 CPC** | 500~5,000원+ |
| **진입 전략** | 특정 세그먼트 집중 (예: "사회초년생 재테크", "프리랜서 세금 가이드") |

### 추천 2: "IT 도구 & 생산성 리뷰 사이트"

**컨셉**: AI 도구, SaaS, 앱, 생산성 도구, 코딩 교육 등 IT/테크 관련 비교 리뷰 및 사용 가이드 사이트.

| 구분 | 내용 |
|------|------|
| **장점** | - AI/테크 분야 검색 수요 폭발적 증가 |
| | - 중간 수준의 경쟁 (개인 블로거가 경쟁 가능) |
| | - 에버그린 + 트렌드 콘텐츠 믹스 가능 |
| | - 글로벌 확장 용이 (영문 콘텐츠 병행 가능) |
| | - 소프트웨어 제휴 마케팅 추가 수익 |
| | - 직접 경험/사용 후기로 E-E-A-T 쉽게 충족 |
| **단점** | - 금융 대비 CPC가 낮은 편 |
| | - 기술 트렌드가 빠르게 변해 콘텐츠 수명 짧을 수 있음 |
| | - 일정 수준의 기술적 지식 필요 |
| | - 신규 도구/서비스 지속적 학습 필요 |
| **예상 CPC** | 200~2,000원 |
| **진입 전략** | 특정 분야 집중 (예: "1인 기업 생산성 도구", "AI 콘텐츠 생성 도구 비교") |

### 추천 3: "생활 정보 & 정부 지원금 가이드"

**컨셉**: 정부 지원금, 생활 꿀팁, 공공서비스 이용법, 자격증 정보 등 실생활에 필요한 정보를 총망라하는 사이트.

| 구분 | 내용 |
|------|------|
| **장점** | - **가장 낮은 진입 장벽** (전문 지식 없이도 시작 가능) |
| | - 매우 높은 검색량 (정책 변경 시 대량 트래픽) |
| | - 콘텐츠 작성이 비교적 쉬움 (공공 정보 기반) |
| | - AdSense 승인 받기 가장 쉬운 분야 |
| | - 다양한 롱테일 키워드 활용 가능 |
| | - 시즌별 이벤트성 트래픽 확보 용이 |
| **단점** | - CPC가 상대적으로 낮음 (100~500원) |
| | - 대량의 트래픽이 있어야 의미 있는 수익 |
| | - 정보의 정확성/최신성 유지 필수 |
| | - 유사 사이트가 매우 많음 |
| | - 콘텐츠 차별화가 어려움 |
| **예상 CPC** | 100~500원 |
| **진입 전략** | 특정 대상층 집중 (예: "신혼부부 지원금 총정리", "대학생 장학금 가이드") |

## 4. 종합 추천

### 최종 추천: "실용 재테크 & 개인 금융 가이드" + "IT 도구 리뷰" 하이브리드

**이유**:
1. 재테크 주제로 **높은 CPC** 확보
2. IT 도구 리뷰로 **꾸준한 트래픽** 및 **쉬운 E-E-A-T 충족**
3. 두 주제의 교차점 활용 가능 (예: "재테크에 필요한 AI 도구", "가계부 앱 비교")
4. 한 주제에만 의존하지 않아 **리스크 분산**

### AdSense 승인을 위한 실행 체크리스트

- [ ] 도메인 등록 및 호스팅 설정 (HTTPS 필수)
- [ ] 사이트 기본 구조 구축 (반응형 디자인)
- [ ] 필수 페이지 3종 생성 (소개, 개인정보처리방침, 문의)
- [ ] 권장 페이지 2종 생성 (면책조항, 이용약관)
- [ ] 고품질 콘텐츠 15~20개 작성 (각 1,000단어+)
- [ ] Google Search Console 등록 및 사이트맵 제출
- [ ] 사이트 속도 최적화 (4초 이내 로딩)
- [ ] AdSense 신청
- [ ] 승인 대기 (3~14일)
