무료 템플릿을 다운로드해서 써봤지만 내 생활에 맞지 않아 포기한 경험이 있을 것입니다. 남이 만든 틀에 억지로 맞추다 보면 오래가지 못합니다. 차라리 본인 소비 패턴에 딱 맞는 가계부를 직접 만드는 것이 훨씬 효과적입니다.

이 글에서는 엑셀 또는 구글 스프레드시트로 직장인 맞춤 가계부를 처음부터 만드는 방법을 단계별로 안내합니다.

## 왜 직접 만드는 것이 좋은가

### 무료 템플릿의 한계

인터넷에서 받은 무료 템플릿은 보통 범용적으로 설계되어 있습니다. 자취생에게 필요 없는 "학비" 항목이 있거나, 직장인에게 꼭 필요한 "야근 택시비", "팀 회식비" 같은 세부 항목이 빠져 있습니다.

직접 만들면 다음과 같은 장점이 있습니다.

- 내 소비 패턴에 맞는 카테고리 구성
- 불필요한 항목 제거로 입력 시간 단축
- 월급일 기준 예산 설정 가능
- 원하는 분석 항목 추가 자유

### 필요한 준비물

- **마이크로소프트 엑셀** 또는 **구글 스프레드시트** (무료)
- 최근 3개월간 카드 사용 내역 (카드사 앱에서 다운로드)
- 30분의 초기 설정 시간

## 1단계: 시트 구조 설계

시트를 3개로 나누면 관리가 편합니다.

| 시트 | 용도 | 핵심 내용 |
|------|------|----------|
| 일별 기록 | 매일 수입/지출 입력 | 날짜, 카테고리, 금액, 결제수단 |
| 월별 대시보드 | 한 달 요약 | 카테고리별 합계, 예산 대비 실적 |
| 설정 | 카테고리, 예산 관리 | 드롭다운 목록, 월 예산 설정 |

### 설정 시트 먼저 만들기

설정 시트에 카테고리 목록을 먼저 입력합니다. 직장인 기준 추천 카테고리는 다음과 같습니다.

- **고정비**: 월세, 관리비, 통신비, 보험료, 구독료
- **식비**: 점심(회사), 저녁/외식, 카페, 배달
- **교통비**: 대중교통, 택시, 주유
- **생활비**: 마트/편의점, 생활용품, 의료비
- **여가비**: 문화생활, 운동, 취미
- **쇼핑**: 의류, 전자기기, 온라인쇼핑
- **경조사비**: 축의금, 조의금, 선물
- **저축/투자**: 적금, 펀드, 주식

카테고리는 7~10개가 적당합니다. 너무 세분화하면 분류하기 귀찮아져서 포기하게 됩니다.

## 2단계: 일별 기록 시트 만들기

### 열 구성

| 열 | 항목 | 입력 방식 |
|----|------|----------|
| A | 날짜 | 직접 입력 |
| B | 구분 | 드롭다운 (수입/지출) |
| C | 카테고리 | 드롭다운 (설정 시트 연동) |
| D | 세부 내용 | 직접 입력 |
| E | 금액 | 숫자 입력 |
| F | 결제수단 | 드롭다운 (신용카드/체크카드/현금/이체) |

### 드롭다운 설정 방법

드롭다운을 설정하면 매번 타이핑할 필요 없이 클릭 한 번으로 선택할 수 있습니다.

1. 해당 열 전체를 선택합니다
2. **데이터 > 데이터 유효성 검사** 클릭
3. 허용 조건에서 **목록** 선택
4. 원본에 설정 시트의 카테고리 범위 지정

구글 스프레드시트에서는 **데이터 > 데이터 확인**에서 동일하게 설정합니다.

## 3단계: 핵심 수식 입력

### 월별 지출 합계

```
=SUMIFS(E:E, B:B, "지출", A:A, ">="&DATE(2026,3,1), A:A, "<"&DATE(2026,4,1))
```

### 카테고리별 월 합계

```
=SUMIFS(E:E, C:C, "식비", B:B, "지출", A:A, ">="&DATE(2026,3,1), A:A, "<"&DATE(2026,4,1))
```

### 예산 잔여액 계산

월별 대시보드 시트에서 각 카테고리의 예산 대비 잔액을 자동 계산합니다.

```
=설정시트의예산금액 - 해당카테고리월합계
```

잔여액이 음수가 되면 조건부 서식으로 빨간색 표시를 설정하면 한눈에 초과 여부를 확인할 수 있습니다.

### 저축률 자동 계산

```
=(월수입합계-월지출합계)/월수입합계*100
```

월급 300만원에 지출 220만원이면 저축률은 약 26.7%입니다. 직장인 기준 20% 이상 유지를 목표로 하세요.

## 4단계: 월별 대시보드 구성

대시보드에는 다음 항목을 배치합니다.

### 상단 요약 영역

- **이번 달 수입**: SUMIFS 수식으로 자동 집계
- **이번 달 지출**: SUMIFS 수식으로 자동 집계
- **잔액 (수입-지출)**: 자동 계산
- **저축률**: 자동 계산

### 카테고리별 예산 대비 현황

| 카테고리 | 예산 | 실제 지출 | 잔여 | 달성률 |
|----------|------|----------|------|--------|
| 식비 | 400,000 | 320,000 | 80,000 | 80% |
| 교통비 | 100,000 | 85,000 | 15,000 | 85% |
| 여가비 | 150,000 | 120,000 | 30,000 | 80% |

달성률이 100%를 넘으면 해당 카테고리에서 과소비가 발생한 것입니다.

## 5단계: 차트 추가하기

### 원형 차트

카테고리별 지출 비율을 원형 차트로 만들면 어디에 돈을 가장 많이 쓰는지 직관적으로 파악됩니다. 데이터 범위를 선택한 후 **삽입 > 차트 > 원형**을 선택합니다.

### 막대 차트

예산 대비 실제 지출을 막대 차트로 나란히 비교하면 어떤 카테고리에서 예산을 초과했는지 명확하게 보입니다.

## 직장인 맞춤 활용 팁

### 월급일 기준으로 설계하기

1일~말일 기준이 아니라 월급일(예: 25일) 기준으로 한 달을 설정하면 실제 자금 흐름과 일치합니다. 25일에 월급을 받는다면 25일~다음 달 24일을 한 주기로 설정하세요.

### 점심값 따로 추적하기

직장인 지출에서 점심 식비가 상당한 비중을 차지합니다. 일 1만원씩 22일 근무하면 월 22만원입니다. 점심 카테고리를 별도로 만들어 추적하면 절약 포인트를 찾기 쉽습니다.

### 주간 리뷰 습관 만들기

매일 입력이 부담스럽다면 일주일에 한 번, 주말에 카드 내역을 보고 한꺼번에 정리하세요. 10~15분이면 충분합니다.

## 마무리

직접 만든 가계부는 내 생활에 맞게 언제든 수정할 수 있다는 것이 가장 큰 장점입니다. 처음 30분만 투자하면 이후로는 주 1회 15분 정도의 시간만 들이면 됩니다.

완성한 가계부를 3개월만 꾸준히 기록하면 월급이 어디로 사라지는지 명확하게 보이기 시작합니다. 지출 패턴을 알면 절약할 곳이 보이고, 절약한 돈을 투자로 돌릴 수 있습니다.
