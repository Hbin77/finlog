월급을 받고, 적금을 붓고, 주식에 투자하고, 카드 대금을 내고. 돈이 여러 곳에 흩어져 있으면 "내 자산이 정확히 얼마인지" 대답하기 어렵습니다.

자산 현황을 정확히 파악하는 것은 재테크의 출발점입니다. 현재 위치를 알아야 목표를 세울 수 있고, 전략을 짤 수 있습니다. 이 글에서는 순자산 계산법과 자산을 체계적으로 추적하는 방법을 안내합니다.

## 순자산이란

순자산은 내가 가진 모든 자산에서 모든 부채를 뺀 금액입니다.

```
순자산 = 총자산 - 총부채
```

예를 들어 예적금 2,000만원, 주식 500만원이 있고, 학자금 대출 800만원이 남아있다면 순자산은 1,700만원입니다.

순자산이 플러스라면 자산이 부채보다 많은 건전한 상태이고, 마이너스라면 부채 상환을 우선해야 합니다.

## 자산 항목 정리

자산을 빠짐없이 파악하려면 항목별로 체크리스트를 만들어야 합니다.

### 금융 자산

| 항목 | 확인 방법 | 비고 |
|------|----------|------|
| 예금 | 은행 앱 | 수시입출금, 정기예금 |
| 적금 | 은행 앱 | 만기 일자, 적용 금리 확인 |
| 청약 저축 | 은행 앱 | 납입 원금 기준 |
| 주식/ETF | 증권사 앱 | 현재 평가금 기준 |
| 펀드 | 증권사/은행 앱 | 현재 평가금 기준 |
| 채권 | 증권사 앱 | 매입가 또는 평가금 |
| 연금저축 | 증권사/보험사 앱 | 적립금 확인 |
| IRP | 증권사/은행 앱 | 적립금 확인 |
| 퇴직연금 | 회사 인사 시스템 | DC형은 운용 현황 확인 |
| 보험 해약환급금 | 보험사 앱 | 해약 시 돌려받을 금액 |

### 실물 자산

| 항목 | 평가 방법 |
|------|----------|
| 부동산 | 국토교통부 실거래가, KB시세 |
| 자동차 | 중고차 시세 (엔카, KB차차차) |
| 귀금속 | 한국금거래소 시세 |

### 부채 항목

| 항목 | 확인 방법 |
|------|----------|
| 주택담보대출 | 은행 앱 (잔액 확인) |
| 전세자금대출 | 은행 앱 |
| 학자금대출 | 한국장학재단 |
| 신용대출 | 은행 앱 |
| 카드론/현금서비스 | 카드사 앱 |
| 카드 할부 잔액 | 카드사 앱 (미결제 할부금) |

## 순자산 계산 실전 예시

20대 후반 직장인 A씨의 자산 현황을 예시로 계산해 봅니다.

### 자산

| 항목 | 금액 |
|------|------|
| 수시입출금 통장 | 3,500,000원 |
| 정기 적금 (2건) | 12,000,000원 |
| 청약 저축 | 4,800,000원 |
| 주식 (국내+해외) | 8,500,000원 |
| 연금저축펀드 | 3,200,000원 |
| **총 자산** | **32,000,000원** |

### 부채

| 항목 | 금액 |
|------|------|
| 학자금 대출 잔액 | 5,000,000원 |
| 카드 할부 잔액 | 800,000원 |
| **총 부채** | **5,800,000원** |

### 순자산

```
32,000,000 - 5,800,000 = 26,200,000원
```

A씨의 순자산은 약 2,620만원입니다.

## 자산 추적 도구 비교

정기적으로 자산 현황을 추적하면 재무 목표 달성 여부를 확인할 수 있습니다.

### 1. 뱅크샐러드

**자산 추적에 가장 적합한 앱**입니다. 은행, 카드, 증권, 보험, 연금까지 거의 모든 금융 계좌를 자동 연동하여 순자산을 한눈에 보여줍니다.

- 장점: 자동 연동, 순자산 변화 그래프, 카테고리별 자산 비중
- 단점: 부동산, 자동차 등 실물 자산은 수동 입력 필요
- 비용: 기본 무료

### 2. 토스

토스도 자산 통합 조회 기능을 제공합니다. 뱅크샐러드만큼 상세하지는 않지만, 이미 토스를 쓰고 있다면 별도 앱 없이 확인 가능합니다.

- 장점: 간편한 UI, 별도 설치 불필요
- 단점: 자산 분석 기능이 뱅크샐러드보다 제한적
- 비용: 무료

### 3. 구글 스프레드시트 (직접 관리)

앱에 금융 정보를 연동하는 것이 불안하다면 직접 스프레드시트로 관리하는 방법이 있습니다.

**추천 구성**:

| 시트 | 내용 |
|------|------|
| 자산 현황 | 항목별 잔액 입력, 총자산/순자산 자동 계산 |
| 월별 추이 | 매월 순자산 기록, 증감 확인 |
| 자산 배분 | 현금, 주식, 채권, 연금 비중 계산 |

매월 1일에 10분만 투자해서 각 계좌 잔액을 업데이트하면 됩니다.

### 4. 계좌정보통합관리서비스 (어카운트인포)

금융감독원에서 운영하는 공식 서비스입니다. 본인 이름으로 개설된 모든 금융 계좌를 한 번에 조회할 수 있습니다.

- 장점: 공신력 있는 정부 서비스, 잊고 있던 계좌 발견 가능
- 단점: 실시간 잔액 조회는 불가, 계좌 유무만 확인
- 비용: 무료

## 자산 배분 점검

순자산을 파악한 후에는 자산이 어디에 얼마나 분배되어 있는지 점검해야 합니다.

### 20~30대 직장인 권장 자산 배분

| 자산 유형 | 권장 비중 | 역할 |
|----------|----------|------|
| 비상금 (수시입출금) | 10~15% | 최소 월 생활비 3~6개월분 |
| 안전 자산 (예적금, 채권) | 20~30% | 원금 보장, 안정적 이자 |
| 성장 자산 (주식, ETF) | 30~50% | 장기 자산 증식 |
| 연금 (연금저축, IRP) | 10~20% | 노후 대비, 세액공제 |

비상금 없이 주식 비중이 80%라면 시장 하락 시 생활비 마련을 위해 손절해야 하는 상황이 올 수 있습니다. 비상금을 먼저 확보한 뒤 투자 비중을 늘려가세요.

## 월별 순자산 추적 방법

### 매월 1일 업데이트 루틴

1. 뱅크샐러드 또는 토스에서 총자산 확인 (3분)
2. 부채 잔액 확인 (2분)
3. 순자산 기록 (1분)
4. 전월 대비 증감 확인 (1분)
5. 자산 배분 비중 점검 (3분)

총 10분이면 월 1회 자산 점검이 완료됩니다.

### 순자산 증가 목표 세우기

현재 순자산에서 연간 얼마를 늘릴 것인지 목표를 세우면 동기부여가 됩니다.

예시: 현재 순자산 2,600만원인 직장인이 매월 100만원씩 저축/투자하면

- 1년 후: 약 3,800만원 (저축 + 투자 수익)
- 3년 후: 약 6,200만원
- 5년 후: 약 9,000만원 (연 수익률 5% 가정)

## 마무리

자산 현황 파악은 재테크의 기본 중의 기본입니다. 현재 순자산을 정확히 알아야 구체적인 재무 목표를 세울 수 있고, 매월 추적하면 목표까지의 진행 상황을 확인할 수 있습니다.

오늘 바로 할 수 있는 일은 뱅크샐러드나 토스에서 모든 계좌를 연동하고 순자산을 확인하는 것입니다. 10분이면 충분합니다. 자산 현황을 한 번 파악하면 돈을 대하는 태도가 달라집니다.
